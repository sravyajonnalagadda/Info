To give access or add a new port

sudo firewall-cmd --zone=public --add-port=8080/tcp --permanent
sudo firewall-cmd --reload

validation:
sudo firewall-cmd --list-ports

command to remove port:
sudo firewall-cmd --remove-port=<port_number>/tcp --permanent
sudo firewall-cmd --reload

Command to delete docker images for disk space:

docker image prune -a --filter "until=$(date -d '8 weeks ago' +%Y-%m-%dT%H:%M:%S)"
